# STC8NeoPixel  
  
Simply example project commanding an RGBW NeoPixel strip with an STC8G1K08A microcontroller.

All it currently does is show a red, green, blue, white pattern on a 9-pixel strip with varying brightness.

## Binaries

STC ISP programming software (v6.91): 
- https://www.stcmicro.com/stc/stc8g1k08a.html

Or [`pip install stcgal`]:
- https://github.com/grigorig/stcgal

SDCC 4.5.0: 
- https://sdcc.sourceforge.net/

CMake & IDE (QtCreator Community):
- https://www.qt.io/
